# momso

momso is the umbrella SaaS product in this repository.

The old 3D anatomy direction is deferred. That work is now treated as a parked prototype for a future sister product under the working name Body Note.

## Current Baseline

- Repository identity: `momso`
- Cloud project identity: momso-first
- Active setup focus: auth, database, deployment, workspace persistence, and product-direction flexibility
- Deferred product concept: Body Note, a future 3D anatomy/education tool
