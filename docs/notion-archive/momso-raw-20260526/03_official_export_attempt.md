# 노션 공식 워크스페이스 export 시도 기록

작성일: 2026-05-26

## 진행한 일

- 노션 웹에 로그인된 상태에서 `Blue Boy` 워크스페이스 설정을 열었다.
- `설정과 멤버 > 워크스페이스 > 일반 > 내보내기 > 워크스페이스 콘텐츠`로 이동했다.
- 내보내기 형식은 기본값인 `Markdown & CSV`였다.
- `내보내기`를 실행했다.

## 화면에 표시된 상태

노션은 다음 메시지를 표시했다.

```text
내보내기 진행 중...
다운로드 링크가 포함된 이메일도 보내 드립니다.
```

2026-05-26 추가 확인:

- 사용자가 Notion 로그인을 완료한 뒤 같은 화면에서 export 진행 상태를 다시 확인했다.
- 약 3분 이상 대기했지만 화면은 여전히 `내보내기 진행 중...` 상태였다.
- 로컬 `Downloads` 폴더에는 Notion export ZIP 파일이 아직 생성되지 않았다.

2026-05-26 재확인:

- 몇 시간 뒤 다시 확인했을 때 `내보내기 진행 중...` 팝업은 더 이상 표시되지 않았다.
- 워크스페이스 설정의 `내보내기` 버튼은 다시 보이는 상태였다.
- 로컬 `Downloads` 폴더와 최근 생성 파일 검색에서는 Notion export ZIP을 찾지 못했다.
- 따라서 완료 파일은 브라우저 직접 다운로드가 아니라 계정 이메일(`eastshine97@uos.ac.kr`)로 발송된 링크를 통해 받아야 할 가능성이 높다.

## 제한

Codex 인앱 브라우저는 직접 파일 다운로드 이벤트를 지원하지 않아, 브라우저 안에서 ZIP 파일을 바로 받을 수 없었다. 따라서 완료된 export 파일은 노션 계정 이메일로 발송된 다운로드 링크를 통해 받아야 한다.

현재 Notion 연결에서 확인된 계정 이메일:

```text
eastshine97@uos.ac.kr
```

## 다음 단계

1. `eastshine97@uos.ac.kr` 메일함에서 Notion export 메일을 확인한다.
2. 다운로드 링크로 ZIP 파일을 받는다.
3. ZIP 파일을 `docs/notion-archive` 아래에 날짜별 폴더로 풀어 정리한다.
4. `몸소`, `빅블루`, `모두의창업`, `Body Note`, `바디노트`, `바디카빙` 등으로 로컬 검색 인덱스/목록을 만든다.
