# momso API v0 Draft

상태: draft
작성일: 2026-06-02

## 목적

PoC 단계에서 momso 웹앱, 외부 저장소, 음성 전사, 수련생 리포트, 외부 AI 도구가 연결될 수 있도록 HTTP API 표면을 미리 정의한다.

현재는 구현 완료 문서가 아니라 설계 초안이다.

## 인증 원칙

- 6/12 MVP: 로그인 없이 데모 URL 공개
- PoC: Naver Login 또는 별도 인증 도입 검토
- API 호출은 사용자/스튜디오/수련생 권한을 분리한다.
- 원본 녹음 접근은 기본 차단한다.

## 엔드포인트 후보

### `GET /api/health`

서비스 상태 확인.

응답 예시:

```json
{
  "ok": true,
  "service": "momso",
  "surface": "prototype"
}
```

### `GET /api/v1/classes/:classId`

수업 맥락 조회.

포함 후보:

- 수업명
- 시간
- 강사
- 참여 수련생
- 동의 상태
- 최근 InBody 참고 지표
- 다음 수업 관찰 포인트

### `POST /api/v1/recordings/intake`

원본 녹음 intake metadata 등록.

주의:

- 원본 파일 자체를 API 요청 body에 직접 넣지 않는다.
- PoC에서는 Object Storage object key 또는 업로드 세션만 기록한다.
- 원본 녹음은 기본 비공개다.

요청 예시:

```json
{
  "classId": "class_20260602_hatha_basic",
  "storageProvider": "ncp_object_storage",
  "objectKey": "studios/bigblue/classes/20260602/recording.wav",
  "consentScope": ["recording", "transcription", "report"]
}
```

### `POST /api/v1/transcripts/draft`

전사/요약 초안 생성 요청.

주의:

- AI 결과는 `draft` 상태로 시작한다.
- AI는 공유 여부를 확정하지 않는다.
- 요가 용어, 타인 정보, 민감 대화는 확인 필요로 표시될 수 있다.

### `GET /api/v1/review-items?classId=:classId`

AI 초안과 강사 검수 상태 조회.

응답 예시:

```json
{
  "classId": "class_20260602_hatha_basic",
  "items": [
    {
      "id": "review_grounding",
      "text": "오른발 접지와 내쉬는 호흡에서 골반 정렬을 함께 확인했습니다.",
      "status": "draft",
      "suggestedStatus": "shareable",
      "needsTermReview": false
    }
  ]
}
```

### `PATCH /api/v1/review-items/:itemId`

강사 검수 상태 확정.

요청 예시:

```json
{
  "status": "shareable",
  "reviewedBy": "teacher_kim",
  "reason": "수련생 개인 리포트에 포함 가능"
}
```

### `POST /api/v1/reports/publish`

검수된 항목만 수련생 리포트로 발행.

가드:

- `shareable` 항목이 없으면 발행하지 않는다.
- `hold`, `internal`, `excluded`, `draft`는 리포트에 포함하지 않는다.

### `GET /api/v1/reports/:reportId`

수련생 리포트 조회.

주의:

- 원본 녹음 URL을 포함하지 않는다.
- 전체 전사본을 기본 포함하지 않는다.
- 인바디 참고 지표에는 진단/처방 아님 문구를 포함한다.

### `GET /api/v1/student-archive/:studentId`

수련생 개인 아카이브 조회.

PoC 이후 후보:

- 수업별 리포트
- 감각 메모
- 다음 수업 연결 포인트
- 외부 AI/API 연결 동의 상태

## 상태 모델

```text
draft -> shareable -> published
draft -> internal
draft -> hold
draft -> excluded
```

## 아직 구현하지 않을 것

- 원본 녹음 파일 공개 다운로드
- AI 초안 자동 발행
- 의료 진단/처방 API
- 수련생 SNS 전체 피드
