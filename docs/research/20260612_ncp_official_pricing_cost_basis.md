# NCP 공식 STT 단가 확정과 변동비 700원 기준 — CLOVA 정가는 수업당 1,680원, BM은 벤더 중립 전사 시장가 700원으로 재산정

> TL;DR: NCP CLOVA Speech 공식 단가를 확정한 결과 정가 기준 60분 수업당 STT 변동비는 1,680원으로 기존 가정(600원)을 크게 초과한다. 이에 BM 원가 기준을 벤더 중립 전사 시장가(보수 상단) 기준 수업당 약 700원으로 교체했고, 새 기준에서도 단위경제(GM 57~72%)와 J커브 좌표는 유지된다.

- 작성일: 2026-06-12
- 질문: 미결이던 CLOVA Speech·Object Storage 정확 단가를 공식 출처로 확정하고, 그 결과가 수업당 변동비 기준(기존 600원)을 어떻게 바꾸는지 확인한다.

## 배경

몸소 BM의 핵심 변동비는 수업 녹음의 STT(음성 인식) 처리 비용이다. 기존 단위경제 모델은 STT를 분당 5~8원, 수업당 300~500원으로 가정해 수업당 변동비 600원 기준을 써 왔다. 그러나 NCP(네이버 클라우드 플랫폼) CLOVA Speech와 Object Storage의 정확한 정가가 미결 상태여서, 이 가정의 성립 여부를 공식 출처로 검증할 필요가 있었다.

## 방법

NCP 공식 상품 페이지의 요금 예시 텍스트("10초 이용 시 5원, 32초 이용 시 15원 과금")와, 페이지가 호출하는 공식 가격 API 응답(CLOVA Speech 장문 인식·Object Storage 상품코드)을 교차 대조해 일치를 확인했다. 전사 시장가 비교군은 주요 전사 API 공급사의 공개 가격 페이지를 사용했다.

- 기존 가정(STT 분당 5~8원, 수업당 300~500원) 대비 **3.5~5.6배** — CLOVA 정가 기준으로는 기존 변동비 600원이 성립하지 않는다.

### 영향 — CLOVA 정가 유지 시 BM 붕괴

- 수업당 변동비 1,680 + Claude 약 160 + 저장 약 2 ≈ 1,850~1,900원
- 프로(9.9만, 45회): GM 약 15% / 맥스(19.8만, 120회 풀 사용): 적자
- 업장 월 변동비(80수업) 약 15만 → 연 기여이익 음수, BEP 불성립

### 채택한 기준 — 전사 시장가 700원

본 산출은 벤더 중립 "전사 엔진(교체형 모듈)" + 전사 API 시장가를 기준으로 하고, PoC 기간 STT는 NCP CLOVA Speech 적용(스폰서 제휴 협의 대상)으로 각주 처리한다. NCP 제휴 요청과 정합 — 정가 기준 수업당 1,680원이라는 사실 자체가 제휴 단가 협의의 근거가 된다.

| 구성 | 수치 | 출처 |
|---|---|---|
| STT 시장가 | 분당 $0.003~0.006(4.5~9원, 1,500원/$) × 60분 = 270~540원 | OpenAI API 가격 |
| STT 보조(화자 분리 포함 증빙) | 시간당 $0.15~0.21 + 화자 분리 $0.02 = 분당 약 4.3~5.8원 | AssemblyAI 가격 |
| Claude 요약 | 입력 2만 + 출력 3천 토큰 = $0.105 ≈ 160원 | Claude API 가격 |
| 저장 | GB당 28원 → 월 1~2원 | NCP Object Storage |
| CLOVA(PoC·참고) | 분당 28원(음성 20 + 화자 8) | NCP CLOVA Speech |
| **수업당 변동비** | **≈ 700원** (보수 상단: 540 + 160 + 2) | — |

참고: Deepgram은 사전녹음 Nova-3 분당 $0.0077부터로 4.5~9원 띠 상단을 벗어나 출처에서 제외. Q&A 대응 — "주요 전사 API 하위 가격대 기준, 산출은 보수 상단(540원)".

### 새 단위경제

- 프로: 45회 × 700 = 31,500 → GP 67,500 → **GM 68.2%**
- 맥스 평균(80회): 56,000 → GP 142,000 → **GM 71.7%** / 헤비(120회): 84,000 → GP 114,000 → **GM 57.6%**
- BYOK Δ 정합: 프로 Δ3만 ≈ 이전 원가 3.15만, 맥스 Δ5만 ≈ 평균 사용 이전 원가 5.6만
- J커브 좌표 무영향: 업장 월 변동비(80수업) 5.6만 ≤ 모델 가정 C=6만(보수) → BEP 23업장·기여이익 88.8만·저점·+44억 전부 유지

## 몸소에의 함의

- 변동비 기준선을 600원에서 700원(벤더 중립 전사 시장가, 보수 상단)으로 교체하는 것이 정당하며, 제출용 BM·단위경제는 이 신 기준을 따른다.
- CLOVA 정가(수업당 1,680원)를 그대로 쓰면 BM이 붕괴하므로, STT는 교체형 모듈로 두고 PoC 기간 NCP CLOVA는 스폰서 제휴 협의 대상으로 둔다. 정가가 높다는 사실 자체가 NCP 제휴 단가 협의의 레버리지가 된다.
- 신 기준에서도 단위경제(GM 57.6~71.7%)와 J커브 좌표(BEP 23업장 등)가 모두 유지되어 BM 서사에 균열이 없다.
