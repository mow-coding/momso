# OKO 마케팅·바이럴 문법 분석 — 같은 제품이라면 승부는 팀과 마케팅 DNA에서 갈린다

> TL;DR: OKO의 마케팅은 "창업자 1인 퍼스널 브랜딩 × 물량전 × 알고리즘·FOMO 해킹"으로 빠르고 싸고 시끄럽다. 강점은 속도·진정성·노출이고, 약점은 신뢰의 깊이·정밀 타겟팅·검증이다. momso는 정확히 그 약점 축(검증된 신뢰·HITL·정밀 도메인)에서 다르게 싸운다.

- 작성일: 2026-06-18
- 질문: OKO 제품이 momso와 거의 동일하다면, 저쪽이 쓰는 마케팅 수단·방법·바이럴 문법은 무엇이고, momso는 무엇을 차용·회피·역공해야 하는가?

## 배경

OKO(무브히)는 momso와 제품 구성이 거의 겹치는 직접경쟁사다. 제품이 비슷하다면 승부는 제품이 아니라 팀과 마케팅에서 갈린다는 가설에서, OKO 대표가 공개 채널에서 구사하는 마케팅 수단·전술·화법을 정리하고 우리와의 차이를 정의한다. 관찰 대상은 OKO 측의 공개 SNS·유튜브·언론 보도·랜딩 페이지다.

## 방법

OKO의 공개 디지털 흔적을 수집·분석했다.
- OKO 대표 개인 Threads 계정(빌드로그), 회사 인스타그램 본계정과 매장 단위 서브계정, 회사 유튜브 채널.
- 2025-02-04자 보도자료 기반 언론 보도(ZDNet·이데일리·한국경제·시사저널e·매거진806 등 동시 5개 매체).
- 회사 랜딩 페이지(사전예약·리퍼럴 퍼널).

인용은 공개 게시물의 표현을 그대로 옮긴 것이다.

## 발견

**채널 구성**
- Threads(OKO 대표 개인 명의) = 메인 무대. 하루 2~3개, 계정 개설(2024-12) 직후부터 매일 빌드로그.
- 인스타그램: 회사 본계정(@pppp4_official) + 매장 단위 서브계정 약 23개(분산 로컬 마케팅) + 창업자 개인 계정.
- 유튜브(구독 약 2.81만): ① 탁구 기술강좌 = SEO/롱테일 트래픽 엔진(수십만 뷰) ② 무인창업 영상 = 낚시성 제목("하루 30분 월순익 600만") ③ Shorts = 자기서사("2만5천 유튜버에서 사기꾼이 됐다").
- 언론 PR: 보도자료 1건으로 5개 매체 동시 타전.
- 오프라인: 건물주 세미나 발표, 창업박람회, 리멤버 '커넥트' 키워드 영업.

**전술 (공개 게시물 인용)**
1. 물량전(volume-first): "마케팅 어려우면 스레드 인스타에 글 수십개씩 올리면 되지. 그래도 안되면 수백개. 그래도 안되면… 또륵", "스레드에 글이 바이럴 될 때까지 한다".
2. 게릴라 콜드 DM 스케일: "100만명에게 DM. 2시간 동안 150명 디엠" — 유명 러닝앱 성장법을 그대로 벤치마킹.
3. 알고리즘 해킹·매일 카운트다운: "오늘도 스레드 알고리즘을 타기위해 서비스 개발 글 남김 D-22", "이래도 알고리즘에 안뜰거야?" — D-23→D-0 매일 카운트다운으로 노출 누적.
4. build-in-public(공개 빌드): 개발 과정·실패·팀 고민을 날것으로 공개("매번 틀리지만 같이 고민할 팀이 있어 다행"). 진정성을 바이럴 연료로 사용.
5. 트렌드/논란 올라타기: 'AI 카피앱' 이슈가 뜨자 "어차피 바이럴 이것도 바이럴. 우리 오코도 바이럴에 올라타도록" — 남의 화제를 자기 노출로 전환.
6. SNS 구인·팀빌딩의 마케팅화: "2주만 도와줄 스친 찾고있어. 댓글이나 디엠줘", 개발자 공개 모집 → 구인 글이 곧 노출.
7. FOMO·희소성 퍼널: 랜딩 "1기 선착순 100명·평생 할인·잔여 18석·D-0" + Threads "사전예약 100명 선착순, 무료체험+80% 할인 평생 제공" + 사회적 증명(베타 100+회·후기 도배).
8. 리퍼럴 루프: 가입 무료 100분 + 친구초대 1명당 +100분(최대 300분) = 회원이 코치를 끌어오는 구조.
9. 오프라인 영업: 건물주 세미나(강남, 30분 발표 → 차주 미팅), 리멤버 커넥트 키워드 다이렉트 영업("무료 마케팅 PoC").

**바이럴 문법 (톤·화법)**
- 40대 타깃 자기계발형 화법 + "슴다"체("일이 있었슴다") = 친근·캐주얼.
- 자기서사·약자 서사: "사기꾼이 됐다", "매번 틀리지만", "또륵" — 결함을 드러내 공감을 유발하는 취약성 마케팅.
- 책임감·대의 서사: "자영업 폐업 100만 시대… 가맹사업 하는 나로서 책임감", "코치·선생님 퇴근 빨리 시켜주고 싶다".
- 숫자·이모지·과장: "100만명에게", 로켓 이모지, "D-day".
- 권위 차용: 청년창업사관학교 합격·국가대표 출신·탁구계 거물 팔로우(협회장·국가대표 선수·WTT)로 제도권 후광.

**OKO 방식의 약점·리스크**
- 노이즈 ≠ 신뢰: 물량·FOMO·과장은 후킹엔 강하나 장기 신뢰 앵커링엔 약하다. "사기꾼" 자칭·카피 논란은 양날.
- 오너 1인 의존: 채널이 대표 개인에 묶여 확장·승계 한계, 번아웃 리스크.
- 정밀 타겟 부재: 대규모 콜드 DM은 분모 키우기로, 전환율·정밀도가 낮고 단시간에 DM 차단을 당했다.
- 검증 공백: "매출 2.4배"(공개된 정황상 외부 벤더의 과거 성과로 보임)·"자체개발"·일부 제휴 주장을 검증 없이 인용 → 들키면 신뢰 타격.
- 제품 미수반: 시끄럽게 알리는 데 비해 앱 출시가 늦어(반려 이력) 노출과 실물의 간극이 역효과를 낼 수 있다.

## 몸소에의 함의

같은 제품처럼 보여도 마케팅 DNA가 정반대다. 축별 대비는 다음과 같다.

| 축 | OKO | momso(김성균·유동환) |
|---|---|---|
| 마케팅 주체 | 창업자 1인 퍼스널 브랜딩 | 제품·도메인(유동환 요가 권위)+기술 서사 |
| 핵심 전술 | 물량·콜드DM·알고리즘 해킹 | 귀납적 — 내가 먼저 쓰고 사용자 차이 포섭 |
| 톤 | 자기과시·FOMO·과장·자기서사 | 신뢰·진실·검증된 수치 |
| KPI | 후킹(노출·다운로드·사전예약 수) | 앵커링(리텐션·재사용) |
| 신뢰 근거 | 사회적 증명·권위 차용(다수 미검증) | HITL 검수·민감정보 책임·자체 검증 데이터 |
| 확장 | 수평(다종목)·물량 | 수직 깊이(요가→수련 아카이브)·데이터 모델 |
| 리스크 | 노이즈·과장 들킴·오너 의존 | 속도·노출 부족(저쪽 강점이 우리 약점) |

행동지침:
- 차용할 것: ① build-in-public(우리도 깃·유튜브 공개 타임스탬프 = prior-art 겸 마케팅) ② 리퍼럴 루프 ③ 도메인 권위 차용(유동환=요가 신뢰자본) ④ 데모 시연 영상.
- 회피할 것: 물량 콜드DM·FOMO·과장·자칭 서사. 우리 브랜드 톤(신뢰·진실)과 충돌한다.
- 역공 포인트: 저쪽의 검증 공백과 미수반 제품을 momso의 검증된 자체 수치 + 실제 작동 데모로 대비 → "시끄러운 쪽 vs 진짜 되는 쪽" 구도. 저쪽 강점(탁구 도메인·오프라인 가맹·노출량)과는 정면 충돌을 피하고, 요가·수련 깊이 + SW/HITL이라는 다른 링에서 싸운다.
