# 경쟁사 OKO(movehee) 인텔 — 같은 카테고리, 더 얕은 버전. 위협은 진짜지만 같은 제품은 아니다

> TL;DR: OKO는 momso의 코어("레슨 끝→AI 기록→코치·회원 전달, 코치 구독·회원 무료")를 먼저·더 넓게(8개 종목 수평)·전자동으로 출시했다. 단 강사 검수(HITL)·도메인 깊이·데이터 주권·동의 아키텍처가 없는 얕은 버전이다. OKO의 등장은 카테고리가 실재함을 증명 — 이제 "왜 OKO 말고 momso냐"는 100% 들어올 질문이 됐다.

- 작성일: 2026-06-17
- 질문: 직접 경쟁 서비스 OKO(movehee)의 제품·BM·출시 현황·창업자/회사 역량을 확인하고, momso의 방어 가능한 차별점이 무엇인지 정리한다.

## 배경
아가르스튜디오 사장님이 "몸소가 이미 있었네"라며 OKO를 공유한 것이 계기다. OKO는 2026-06-08 사전오픈한, momso와 코어 아이디어가 겹치는 직접 경쟁 서비스다. 프라이머·인바디라이크 등 외부 심사에서 "왜 OKO가 아니라 momso냐"는 질문이 반드시 나올 것이므로, 그에 대비해 OKO의 실체를 정리한다.

## 방법
전 출처가 공개 페이지다. OKO 랜딩 페이지(movehee.com), 창업자 공개 SNS(Threads) 빌드로그, 공식 인스타그램·유튜브 채널의 공개 게시물, 공개 웹/언론 검색을 종합했다. 미수집분(일부 네이버 블로그·지도)은 분석에 결정적이지 않다.

## 발견

### 1. OKO 제품 (movehee.com)
- 슬로건: **"레슨이 끝나는 순간, AI가 전부 기록합니다."** 코치 기록노동 0분, 회원 매주 성장 확인.
- 작동: 코치가 앱에서 회원 선택 → 녹음 ON → 레슨 → **AI 자동 전사·분석(평균 90초): 핵심 한 줄·다룬 기술·칭찬 횟수·다음 질문 구조화** → 코치(코칭 노트)·회원(성장 일지) 양쪽 앱 푸시. **사람 검수 단계 없음(전자동).**
- 종목: **탁구·테니스·골프·PT·요가·필라테스·수영·복싱 — 모든 1:1 코칭** (수평 확장, 도메인 무관 전사 구조).
- 회원 기능: 매 레슨 AI 성장일지, 코칭 히스토리 영구 보관, 다음 레슨 추천 질문, 12주 누적 성장 리포트.
- 데이터/프라이버시: "모든 녹음 암호화 저장, 분석 후 **자동 삭제 옵션**, 회원 동의 없이 외부 공유 안 함" — 한 줄 수준(momso의 3중 동의 방어 대비 얕음).
- 완성도: "디자인 시안 아닌 실제 작동 앱", 100+회 베타, iOS·Android 출시 준비 완료. movehee.com에 실서비스 로그인 존재.
- 주장 ROI(코치): 하루 30분×월 12h×연 144h 절약. 재등록률 +30%·지인추천 +24%·이탈 -42%·월매출 +20%.

#### BM (momso와 거의 동일 + 차이)
- **코치·관장 구독, 회원 무료** (= momso 강사앱 과금·회원 무료와 동일 구조).
- 사전예약: 코치·관장 **100명 한정(82/100 완료)**, 사전예약자 평생 할인. 가입 즉시 무료 100분, 친구초대 +100분(최대 3명·300분). 6/8 사전오픈 → 6월 중순 정식.
- 가격: 명시 숫자 없음. 베타 코치 후기 "회원들이 **회당 5천원** 정도면 받아볼 의향". (공개 블로그 인터뷰 기준 요금 언급: 400분/3000분 단위.)
- "기록 = 영업 = 재등록 = 매장 생존" — momso와 동일한 핵심 명제.

#### 후기에서 드러난 약점(momso 차별점의 근거)
- "운동 명칭 인식이 **살짝 틀릴 때 있다**"(트레이너), "동작 명칭 인식 틀림" — **전자동의 환각/오인식 리스크가 그대로 노출**. momso의 강사 검수 게이트가 정확히 겨냥하는 지점.
- 도메인 용어 약함(범용 전사) — 종목별 깊이 부족.

### 2. 창업자·회사
- **OKO 대표**: 전 청소년 탁구 국가대표 · 10년차 코치 · 청년창업사관학교 16기 지원. "매번 회원 레슨 내용 기억 못해 스트레스 → 기록부터 성장까지 자동화 개발." 공개 SNS(Threads) 팔로워 약 900, 최근 조회 수만 단위.
- 회사: **탁구발전소24 / movehee**(인천 서구 소재). 개인정보보호책임자는 대표 본인.
- 본업(OKO의 토대): **무인탁구장 가맹사업(매장 30호점 ing)** + 건물 **공실 솔루션** + 아파트 커뮤니티 탁구장 위탁운영 + **탁구 코치 매칭 플랫폼**. 자칭 "탁구 전문 토탈 플랫폼".

- 정부지원/네트워킹(공개 SNS, 2026-01~02): 딥테크·초기창업패키지·청창사 동시 지원. 영업 노하우(다이렉트 영업·무료 마케팅 PoC), 건물주 세미나 발표.
- **해석**: OKO는 신생 단독 스타트업이 아니라, **이미 오프라인 탁구 인프라(무인장 30호점)·코치 매칭 플랫폼·2.8만 유튜브 채널을 가진 운영자가 그 위에 얹은 AI 기록 레이어**다. 유통·코치풀·자금(정부지원)·영업력이 이미 있음 = 만만치 않음.

#### OKO의 우위(직시)
선출시 + 실앱 + 베타검증 + **기존 탁구 유통망/코치풀/채널** + 수평 TAM + 정부지원·영업력.

#### momso의 방어 가능 차별점
1. **안전(HITL 검수)** — 건강·신체 텍스트 환각은 사고. OKO는 전자동이라 이 리스크를 안고 감(후기로 이미 노출).
2. **요가 도메인 깊이** — 범용 전사 vs 요가 용어 딕셔너리·초점 설계.
3. **데이터 주권(BYOK/BYOC)** + **동의 아키텍처** — 민감 데이터 신뢰.
4. **정량 결합(인바디)·플랫폼 비전(패스)**.

### 4. "왜 OKO가 아니라 momso냐" — Q&A 답안

## 몸소에의 함의
- **카테고리 검증**: OKO의 존재 자체가 "레슨→AI 기록→코치·회원 전달, 코치 구독" 시장이 실재함을 증명한다. 심사에서 "이미 OKO가 있는데?"는 위협이 아니라 시장 존재 증거로 되받을 수 있다.

- **비대칭 우위**: OKO의 강점은 유통·코치풀·영업·자금(오프라인 탁구 인프라). 약점은 SW/AI 역량(비개발자 대표·CTO 1인 의존·소수 부트스트랩). momso의 강점이 정확히 OKO의 약점 위에 선다.
