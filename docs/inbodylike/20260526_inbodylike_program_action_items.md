## 발표자료 구성 우선순위

회의에서는 스케일업보다 솔루션과 인바디 활용 시나리오의 비중을 더 높여야 한다고 정리했다.

| 파트 | 역할 | 비중 |
|---|---|---|
| Problem | 요가·라이프스타일 현장에서 체성분 데이터 활용 맥락이 부족하다는 문제 제기 | 중요 |
| Solution | 수업 언어, 피드백, 지도자 의도, 수련생 반응을 기록하는 몸소의 기록 계층 | 가장 중요 |
| InBody 협업 시나리오 | 인바디 기기를 요가원에 도입했을 때 실제 수업 전후로 어떻게 쓰이는지 | 가장 중요 |
| Scale-up | 시장 규모, 제휴 업장 수, 매출 전망, 확장 전략 | 보조 근거 |
| Team | 현장 도메인 지식과 실행력 | 중요 |

회의 표현으로는 `스케일업은 교과 점수`, `솔루션·프로그램 설명은 자기소개서`에 가깝다. 즉 시장 규모 숫자는 필요하지만, 당락을 가르는 핵심은 "인바디 데이터를 이 팀이 현장에서 어떻게 살릴 수 있는가"다.

## 발표자료에 바로 반영할 문장

> 우리는 인바디라이크 1번 라이프스타일 분야에 지원한다. 몸소는 요가 수업의 언어, 피드백, 지도자 의도, 수련생의 반응을 기록하고, 이를 인바디의 체성분 데이터와 연결해 웰니스 현장에서 장기적인 변화의 기록 계층을 만드는 서비스다.

> 인바디는 헬스장과 병원 등 정량 측정이 익숙한 공간에서는 강하지만, 요가처럼 몸의 감각과 수업 맥락이 중요한 현장에서는 데이터가 실제 경험으로 연결되는 사용 시나리오가 부족하다. 몸소는 이 빈틈을 현장 도메인 지식으로 채운다.

> 이번 지원서에서 시장 규모는 보조 근거다. 더 중요한 것은 인바디 기기가 요가원에 들어왔을 때 수업 전 측정, 수업 중 언어 기록, 수업 후 리포트가 어떻게 하나의 경험으로 이어지는지를 구체적으로 보여주는 것이다.
