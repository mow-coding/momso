# 2026-05-26 · 몸소 피치 방향 정리 + 시연·PoC 계획

> 한 줄 요약: 몸소를 "요가 기록 앱"이 아니라 **요가 수업의 기록 계층**으로 포지셔닝하고, 인바디와의 파트너십 서사·요금 구조·시연 영상 방향·PoC 검증 항목을 합의했다.

- 참여: 유동환 · 김성균
- 맥락: 인바디라이크 제출(마감 6/12)을 앞두고, 사업계획서와 발표 서사의 중심 메시지를 "기능 설명"에서 "왜 요가 업계에 기록 계층이 필요한가"로 잡고, 시연 영상과 녹음 환경·요금 모델을 정하기 위한 자리.

## 논의

### 포지셔닝 — 기록 계층
- 몸소는 요가 수업 기록만 하는 앱이 아니라, 인바디가 만드는 **신체 스펙 데이터**와 수업 중 발생하는 **언어·피드백 데이터**를 묶어 수련생이 자기 몸과 수련 이력을 계속 조회하게 하는 **기록 계층 서비스**다.
- 핵심 한 문장: "인바디는 스펙을 기록하고, 몸소는 수업을 기록한다. 이 둘이 융합되어야 요가에서 의미 있는 기록 계층이 된다."
- 수련생 관점에서는 자기 몸 상태를 언제든 조회하는 **세컨드 바디**(챗봇), 요가원 관점에서는 수업·고객 소통을 선명하게 하는 기록 인프라.

### 요가 업계 문제 정의
- **비정형성**: 골프·헬스와 달리 요가는 수업 방법론이 체계화돼 있지 않고 요가원마다 방식이 상이하다 — 다채로움이라는 강점이자 구조적 특징.
- **블랙박스**: 수업의 질을 외부에서 객관 비교하기 어렵고, 강사의 카리스마·라포에 평가가 의존한다.
- **근본 원인 = 기록 계층의 부재**: 요가는 체득에 시간이 필요해 수업 중 몸의 움직임에만 집중하고 기록이 남지 않는 관행이 형성됐다. 악의가 아니라 요가 본분에 충실한 결과이며, 역으로 부담 없는 기록 계층이 생기면 체득 과정을 더 잘 도울 수 있다는 역발상.
- 발표는 요가 업계를 공격하지 않고 **내부자의 진정성 있는 문제의식**으로 접근한다.

### 기술 구현 / 데이터 주권
- 이상: 각 요가원 로컬 하드웨어(고사양 SSD·마이크)에서 AI 연산·저장이 이뤄지는 **데이터 주권** 모델.
- 현실: 컴퓨터 없는 요가원도 있고 저장 관리가 안 되는 곳이 대부분이라 난이도가 높다. 현재는 중간 단계로 외부 전사 엔진(STT 벤더)을 활용하고, 가장 쉬운 형태는 중앙 서버 구축 + 고객은 앱만 사용.
- 절충안: **SSD·마이크 세트를 구독 고객에게 렌탈**. KT 모뎀 설치처럼 엔지니어가 직접 방문해 세팅하는 구조 — 현장 방문이 인간적 접점과 현장 데이터 수집의 부가가치를 만든다. 유사 사례로 동남아 그랩(Grab)의 직접 방문 계약 성장이 언급됨.

### 파트너십 전략 (인바디)
- 하드웨어 파견·설치·고용은 인바디 등 협력사가, 몸소 팀은 도메인 지식 기반 현장 검증 원리 설계와 서비스 매뉴얼에 집중하는 역할 분담.
- 인바디에게는 "우리 앱 써달라"가 아니라 "현장 설치·측정·장비 운영을 같이 맡아달라"는 **역할 제안**이 더 강하다. 인바디를 단순 협찬사가 아니라 요가 시장 진입의 핵심 파트너로 위치시킨다.

### 비즈니스 모델 / 매출 추산
- B2C: 수련생 1인 월 약 5달러(≈7,000원).
- B2B: 요가원 월 30만 원(1년 계약 기준).
- 요가원 1곳당 수련생 20명(보수적) 기준 → B2C 약 15만 원 + B2B 30만 원 = 월 약 45~50만 원, 연 약 600만 원.
- 100곳 도입 시 연 약 6억 원, 1,000곳 도입 시 연 약 60억 원.
- 성장 벤치마크: 유사 플랫폼 오부시(5부시)가 3년 만에 전국 1,000개 업장 달성 → 몸소도 3년 차 연 매출 60억 원 목표.
- 확장: 요가 외 필라테스·PT·골프·테니스로 넓히면 5년 내 매출 300억 원도 가능, 아정당(연 매출 300억 → 1,500억 매각) 수준의 엑싯 시나리오로 연결. SaaS 기업 가치 배수는 통상 8~20배.

### 시연 영상 / 녹음 환경
- 영상 구성: 앱 기능 50% + 실제 사용 흐름 50%. 흐름은 `수업 전 데이터 측정 → 수업 중 실시간 전사 → 수업 직후 피드백`.
- 실시간 전사 시연: 전사 도구 데스크톱을 노트북에서 실행해 수업 중 음성이 **실시간** 전사되는 장면을 촬영(수업 후 요약이 아니라 실시간 처리가 핵심 어필).
- 데이터 통합 연출: 수업 전후 인바디 측정, 애플워치 심박수 데이터를 전사 데이터와 함께 화면에 표시. 인바디 관계자가 "우리 기기가 들어가면 딱 맞겠다"는 상상을 하게 만드는 게 목표.
- 녹음 방식: 특정 기기 강제는 피한다. 목걸이형 녹음기는 동작을 방해하고 특정 기종·고가 요금제로 확장성이 낮다. 옷깃 부착형 소형 블루투스 마이크를 노트북 USB 수신기로 수음하는 방식, 공간 전체 소리를 담는 천장·코너 배치를 실험. 배터리 변수를 고려해 녹음 전담 노트북을 별도 운용.
- 출연: 운영자 배우자와 개발 파트너가 수련생·강사 역할로 연기 가능(실제 수업 허락이 없을 경우 대체).

### 운영 철학
- "잘 가르치는 것"보다 "자신의 의도를 정확히 전달하는 것"이 빅블루 요가원의 유지력·정체성에 가깝다. 남아있는 수련생은 운영자의 의도·진심을 읽어주는 사람들이며, 이 "소통의 전문성"이 앱의 핵심 가치이자 발표 서사의 중심.

## 결정
- 포지셔닝을 "요가 기록 앱"이 아니라 **요가 수업의 기록 계층**으로 확정.
- 발표·사업계획서의 중심을 기능 설명에서 "왜 요가 업계에 기록 계층이 필요한가 / 왜 내부자가 풀 수 있는가"로 이동.
- 시연 영상은 `수업 전 측정 → 수업 중 실시간 전사 → 수업 직후 피드백` 흐름, 앱 기능 50% + 실제 사용 50%.
- 녹음은 특정 기기 강제 대신 "동작을 방해하지 않는 공간형 수음"을 실험하고, 녹음 전담 노트북을 별도 운용.
- 인바디 등 협력사에는 "역할 분담(현장 설치·측정·장비 운영)" 제안으로 접근.
- 기본 요금 단위: 요가원 월 30만 원 + 수련생 1인 월 약 7천 원 → 요가원 1곳 연 약 600만 원, 1,000곳 기준 연 약 60억 원.
- PoC의 목적은 "앱이 작동한다"가 아니라 **요가원마다 다른 현장 디테일(장비 필요성·데이터 공유 범위·수업 스타일별 기록·요금제 구조) 발굴**로 정의.

- 피드백 공간용 가구(무거운 서랍장) 이동 — 추가 인력 필요, 일정 미정.
