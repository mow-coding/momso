# 🎭 6/8 16:00 자문 미팅 플레이북 — 전직 VC · 현직 부동산 중개 선생님

> **상황:** 형식 없는 자유 미팅. 하지만 *아이템 검사*가 목적 → **방향을 우리가 쥐고** 그분 전문성을 최대로 뽑아낸다.
> **이 분의 두 렌즈를 동시에 공략한다:**
> - 🧮 **VC 렌즈** — 시장·단위경제·해자·팀·과장 여부를 *투자 심사하듯* 본다. *과장을 가장 싫어한다.*
> - 🏘 **부동산 중개 렌즈** — 소상공인(=요가원 원장)에게 *파는 법*, 지역 밀집 영업, 신뢰 빠르게 얻는 법에 *현장 직관*. **우리의 약점(§3 BM·§4 영업)을 검증받을 최적임자.**

## 0. 이 미팅에서 우리가 가져갈 3가지 (목표)
1. **VC 렌즈로 구멍 찾기** — 심사자보다 먼저 약점을 듣는다.
2. **부동산/소상공인 렌즈로 영업·가격 검증** — "원장이 월 30만을 *투자*로 볼까?", "밀집 지역 영업 순서?"
3. **구체적 도움 받기** — 그의 네트워크 연결 / VC 시각의 "가장 약한 한 가지" / 60억 화법.

> ⚠️ 미팅 끝에 **§7(다음 액션)**을 꼭 채운다 — "바꿀 것 / 보강 숫자 / 누가·언제까지".

## 1. 오프닝 (60~90초, 통째로 익힐 것)
> "선생님, 형식 없이 편하게 봐주시면 좋겠습니다. 저흰 *요가원을 직접 운영하는* 유동환과 *기술을 맡은* 김성균입니다.
> 한 문장으로 — **'수업에서 사라지는 말·교정·감각을, 강사가 검수해 개인 수련 기록으로 남기는 서비스'** momso입니다.
> 인바디는 *몸의 스펙*을, 저흰 *수업의 맥락*을 기록합니다. 작동하는 데모가 있어서 1분만 보여드리고, **선생님이 가장 의심되는 곳을 찔러주시면** 거기부터 이야기하고 싶습니다."

→ 바로 momso.vercel.app **강사 흐름 1화면.** (말 길게 X, 데모로 보여줌.)

*왜 이렇게:* VC는 *직설·정직·짧음*을 신뢰한다. "찔러달라"고 먼저 열면 그의 전문성이 살고 우리가 방어적으로 안 보인다.

## 2. 자유 미팅의 척추 (흐름 — 외울 순서)
**훅 → 데모 → 문제(킬러) → 해결/해자 → 시장·BM → 정직한 약점 → 그에게 도움 요청**

각 30초 카드:
- **문제:** "변화는 일어나는데 *강사는 보여줄 수 없고 수련생은 볼 수 없다* → 동기·재방문이 끊깁니다." (CRM·예약앱·전사앱·측정기 사이 *빈 칸* = 수업 맥락 기록 계층)
- **해결:** "AI는 *초안만*, **강사가 검수**해 확정한 것만 발행 → 회원별로 쌓입니다(HITL)." 차별점 ①검수+수업맥락(해자) ②BYOK=간편연동(게다가).
- **시장:** "요가(빅블루)에서 *가장 어려운 도메인*을 검증하고 → 인바디 깔린 필라·PT로 확장." TAM 131,764곳·89.9% 소상공인.
- **BM:** "B2B 월 30만은 *고정비가 아니라 재방문 투자*, B2C 7천은 *바이럴*. 기본 3년차 20~26억." (60억은 *조건부*)
- **인바디:** "*첫 정량 데이터 파트너*입니다. 필수가 아니라, 측정값에 *수업의 의미*를 입히는 거예요."

## 3. 🧮 VC 렌즈 — 날아올 질문 & 즉답 (외울 것)
| 그가 물을 것 | 우리 즉답 |
|---|---|
| **"비타민이야 진통제야?"** | 진통제 — *재방문 단절*은 원장 매출 직격. §1 킬러 인사이트. |
| **"누가, 왜 *지금* 돈 내나?"** | 원장(B2B 30만, 리텐션 투자)+수련생(B2C 7천, 바이럴). 1인 다역 소상공인 89.9% + 자기관리 트렌드. |
| **"GPT로 누구나 만들잖아?"** | 기술이 해자가 아님 — **요가 도메인 민감성 검수(HITL)+회원별 축적+현장(동환)**. 회의록 앱엔 구조가 없음. |
| **"시장 작지 않아? 벤처 규모 나와?"** | 기본 20~26억으로 *정직*하게. 확장은 인바디 사용자·필라/PT로. *과장 안 함.* |
| **"왜 너희 둘? 개발자도 없이?"** | *이미 작동하는 데모가 증거*. AI 보조개발로 2인 PoC. 채용은 *투자 후*. |
| **"단위경제? 회수기간?"** | 곳당 월 41~72만(전환 20~50%). BYOK로 *LLM 추론비 안 냄* → 원가↓. 전환율은 PoC 실측 예정. |
| **"60억은 뭐야?"** | *조건부터*: "기본 20~26억, 60억은 1,000곳·유료 30명 가정의 상단." |

> **핵심 태도:** VC 앞에선 **약점을 먼저 인정**하면 신뢰가 오른다. "가장 약한 건 영업입니다"를 *우리가 먼저* 말한다.

## 4. 🏘 부동산/소상공인 렌즈 — 그에게 *물어볼* 것 (이게 이 미팅의 진짜 금맥)
이 분은 *소상공인에게 파는 일*을 매일 한다. 우리의 가장 약한 곳을 *현장 전문가*에게 검증받자.
1. **"소상공인 원장에게 월 30만을 *고정비가 아니라 투자*로 인식시키려면, 현장에선 어떤 말·자료가 먹히나요?"**
2. **"연희동처럼 스튜디오가 밀집한 지역에서 B2B를 까는 *현실적인 영업 순서*는 뭘까요?"** (우리 가설=빅블루 등대→옆집)
3. **"소상공인 영업에서 신뢰를 *빠르게* 얻는 법? 중개하실 때 사장님들 마음 여는 포인트가 뭔가요?"**
4. **"창업자가 직접 영업하다 *언제* 영업 인력을 들이는 게 맞을까요?"** (우리 가설=투자 후)

> 이 질문들은 그의 *자부심*을 건드린다 → 길게 답해주고, 우리 §3·§4가 *현장 검증*된다.

## 6. ⚠️ 지뢰 & 태도
- **과장 금지(최우선).** VC는 냄새 맡는다. 모든 숫자에 *조건*, 모르면 "검증 예정".
- **인바디 과대포장 X.** "필수"가 아니라 "첫 정량 파트너".
- **안 만든 걸 만들었다고 X.** 세컨드 바디 LLM = *PoC 검증 중*.
- **돈 얘기가 나오면(그가 투자/연결 제안 시):** 현금보다 **인맥·동행을 역제안**, 받더라도 *명확한 계약 전제*. (과거 비계약 관계 교훈)
- **부동산 비유 활용:** "발레파킹처럼 — 데이터는 *고객 소유*, 운영은 우리가." (그가 이해하는 언어)
